About The Project

Project Overview

EduMonitor is a comprehensive, AI-powered educational monitoring platform developed at the Sri Lanka Institute of Information Technology (SLIIT). The system addresses a critical gap in modern education — the lack of real-time, actionable insights into classroom dynamics.

Our platform integrates four specialized research components into a unified system: Student Engagement Detection using pose estimation and computer vision, Teacher Behavior Classification through movement tracking and spatial analysis, Attendance Factor Analysis using a validated psychometric survey instrument with machine learning predictions, and a RAG-based Classroom Support System for real-time question answering and lecture summarisation.

Built on a modern tech stack — FastAPI backend, Next.js frontend, and PostgreSQL database — EduMonitor processes classroom video feeds, survey data, and lecture content to deliver real-time analytics, predictions, and personalised recommendations to educators through an intuitive dashboard.

IT22582874
Kisara W.C

Student Engagement Detection

Perception: Real-time 17-point skeletal extraction utilizing YOLOv8-Pose. Tracking: OSNet-based deep re-identification (Re-ID) to handle camera transitions and occlusions. Feature Engineering: Extraction of 9 geometric features for students and dynamic temporal/kinematic features for teachers. Classification: Lightweight Random Forest classifiers employed for low-latency behavioral predictions. Aggregation: Temporal smoothing windows (approx. 30 frames) and majority voting to stabilize predictions.

IT22560162
Deemantha P.H.H.C

Attendance Factor Analysis

The system follows a four-phase pipeline. A 28-item Likert-scale survey covering six factor dimensions was designed based on attendance literature and administered via Google Forms to SLIIT students and students from other Sri Lankan universities. Survey responses were linked to actual IE2051 module attendance records, creating a matched dataset of 115 students. Exploratory Factor Analysis was conducted to validate the six-factor structure, achieving KMO values above 0.90 and Cronbach's alpha between 0.840 and 0.936. Four regression models were compared using five-fold cross-validation. Ridge Regression was selected as the best model with R² = 0.8643. The trained model was integrated into the EduMonitor FastAPI backend as an ML Insights Engine that generates per-student predictions and natural-language recommendations.

IT22594136
Pallewela S.M

RAG Classroom Support

The system uses a dual-source RAG architecture combining teacher-provided materials and live speech-to-text lecture transcripts. Content is embedded using Sentence-BERT and stored in a vector database for retrieval. A fine-tuned language model generates context-aware answers, summaries, and quiz questions based on retrieved information.
